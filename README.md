## 🚀 Verificador de Decolagem do Foguete

## 📖 Explicação do Projeto

Este projeto implementa um simulador de verificação de segurança para a decolagem do Foguete Aurora Siger, desenvolvido em Python. O programa gera valores aleatórios que representam a leitura de sensores de sistemas.

Esses valores são comparados com os valores do **Limite Seguro**, caso todos os parâmetros estejam dentro dos limites definidos, o sistema autoriza a decolagem, caso contrário, será negada  a decolagem.

Após isso é feita uma análise assistir por Inteligência Artificial feita com um modelo de Machine Learn em python, onde ela irá fazer a classificação dos dados, detecção de anomalias e sugestões de riscos.
